#### 新建R语言运行环境

```r
conda create -n r -c r r-essentials
```

#### 删除运行环境

```
`conda remove --name py3 --all    #py3 为运行环境名字
```
#### 安装特定的版本

```
conda install 软件名=版本号
conda install gatk=3.7
```

这时conda会先卸载已安装版本，然后重新安装指定版本。

#### 更新指定软件

```
conda update gatk
```

#### 卸载指定软件

```
conda remove gatk
```

#### 查看已安装软件

```
conda list
```
